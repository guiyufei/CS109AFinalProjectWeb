Visualization Part II: All Baby Names¶

Overview:¶

In this section, we will include the visualization and result discussion for the analysis using all baby names.

(1) The first part of the analysis is using PC1 and PC2 of the cross-corelation matrix to separate the 51 states, and based on the projection of PC1, PC2 to choose the reasonable number of clusters. Then we uses Hierarchical Clustering, K-Means Clustering (related code in Hierarchical Clustering & K-Means Clustering section) to cluster 51 states.

(2) The visulization in this section includes (i) How to choose number of clusters (ii) Using Tableau to reflect the state-clusters in a map view (Tableau may not able to be shown in github, but it can be view on Ipython Notebook, and we will embed Tableau stories in the website)

#Function: corrstate: compute state by state corrleation between the fluctuations of the frequencies of the names over the states
#Input: df: dataframe has states as rows and biblical names as columns for each year
#Return: corr_matrix: matrix contains the Pearson correlation between the flucuations of name in state i and state j
#        for all states in the fluctuation; 
def corrstate(df):
    #sum the count of name by states
    total_sum = df.sum(axis = 1)
    #divide the count of name in each state by the previous sum
    df1 = df.div(total_sum.ix[0],axis='columns')
    statenum = df1.shape[0]
    name_sum = df1.sum(axis = 0)
    #count the average frequency of name over the states
    averfreq = name_sum/float(statenum)
    #use the frequency of name in the state - average frequency of the name over states (he flucuations of name)
    fluctuation = df1.subtract(averfreq, axis ='columns')
    #compute the Pearson correlation among 51 states * 51 states 
    return np.corrcoef(fluctuation)

#Function: PCAcorr: use PC1, 2 to figure out the separate the noise from true system dynamics 
#          Based on the assumption that the PC1, PC2 are related to true similarity
#Input: corr_matrix: Pearson correlation between the flucuations of name over states
#Return: corr_reduced: generated correlation matrix after reducing by PCA

def PCAcorr(corr_matrix):
    corr_matrix = preprocessing.normalize(corr_matrix)
    pca = PCA(n_components=2)
    pca.fit(corr_matrix)
    corr_reduced = pca.transform(corr_matrix)
    return corr_reduced

#Function: Sort PC1 in descending order and assign each state a different color
#          Visualize the PC1,2 distribution for each state
#Input: pc12: Correlation matrix after reducing by PCA (which is based on the first two principle component of biblical names for each state)
#Input: title: the year that the data come from 
#Input: i: column index for data visualization 
#Input: ax_ind: row index for data visualizaion

def plot_pca_year_multi(pc12, title, i,ax_ind):
    # sort pca
    pc12=pc12[np.lexsort(([1,-1]*pc12[:,[1,0]]).T)]
    # split pc1, pc2
    pc1=pc12[:,0]
    pc2=pc12[:,1]
    # Plot data    
    cs = matplotlib.cm.rainbow(np.linspace(0, 1, len(pc12)))
    ax[i][ax_ind].scatter(pc1,pc2,label='pc1 pc2',color=cs)
    # Label axes, set title
    ax[i][ax_ind].set_title(title,fontsize=15)
    ax[i][ax_ind].set_xlabel('$V^{(2)}$',fontsize=15)
    ax[i][ax_ind].set_ylabel('$V^{(1)}$',fontsize=15)
    ax[i][ax_ind].set_xlim(-1,1.5)  
    ax[i][ax_ind].set_ylim(-1,1.5)

PC1 & PC2 visualization

  • We take the first two principle component from PCA for each event (which was calculated based on the matrices that consist of the correlation of distribution of whole baby names in state I and and that in state j). And then we reorder the array (of the PC1, PC2) in the descending order of PC1 and then assign rainbow color.
  • We found that there are 3 clusters (in general, the colors can be separated into three groups) in every PC1/PC2 plots above for 10 matrices. Therefore, we use 3 cluster for hierarchical clustering and K-means clustering to group states. States in the similar groups are going to be assigned similar colors in tableau visualization. The overlap between these points here could be used to describe the similarities movement of populations before and after such event. Instead of considering these overlaps in time, we consider the correlations between states on each event, by studying the whole distribution of baby names in every state.

def generate_table(title,idx):
    n=len(title)
    j=0
    pc=PCA[j]
    df=pd.DataFrame(pc)
    df['idx']=idx
    df = df.set_index(['idx'])
    del df.index.name
    df.columns = ['pc1', 'pc2']
    sorted_df=df.sort('pc1',ascending=False)
    cs = matplotlib.cm.rainbow(np.linspace(0, 1, len(df)))
    color=cs[:,:-1]*255
    row=color.shape[0]
    new_color=[]

    for i in range(0,row):
        a=color[i]
        r=a[0]
        g=a[1]
        b=a[2]
        rgb=65536 * r + 256 * g + b
        new_color.append(rgb)

    YEAR=str(title[j])
    sorted_df[YEAR]=new_color
    table0=pd.DataFrame(sorted_df[YEAR])


    for j in range(1,n):
        pc=PCA[j]
        df=pd.DataFrame(pc)

        del df.index.name

Graphing maps

  • Before using Tableau to generate the colorful US map, we convert the 3 digit RGB color to 1-digit number using a formula. Thus, the color assigned to the states reflect the similarity in their distributions of biblical names.

  • For hieratical clustering and K means clustering results, the three color assigned to the states reflect the similarity in their distributions of names. Visually, their results are almost similar. Whereas, we divide ordered PC1,2 results for 51 states into 3 categories based on values and assign color correspondingly, from which we observe more details. We use a notion of similarity based on the principal component analysis of the matrices of states correlations.

  • The observable trends that we noticed is that FL is the state that change the most after every event we are interested in:

Observed Pattern:

Before and after Alaska become a state: GA and FL changed from similar to different from nearby states like LA, AL, TN, KY.

Before and after Great Depression: TN, GA, AL have been influenced by nearby states

Before and after the second wave feminism (female names): GA, FL, NC changed from being similar to different from nearby states like AL,LA

Before and after the second wave feminism (male names): FL changed from being similar to different from nearby states, like GA, TN, NC.

Before and after the WWII: An infusion of South and nearby states (and California).
